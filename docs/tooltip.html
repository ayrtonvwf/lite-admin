Tooltip

Basic

You can add tooltips to any element you like. Just add a tooltip class to it, and by default the title will be used as the message.



Custom message

If you need to use the element's title for another purpose, and want a custom message for the tooltip, you can override it with the data-tooltip attribute.



Placement

You can change the default placement of the tooltip (top) with the classes tooltip-left, tooltip-right and tooltip-bottom.



Limit Alignment

By default, the tooltip is aligned at the start (top or left) limit of the trigger. You can change this by using the class tooltip-end, so the tooltip will be aligned at the end (bottom or right) limit of the trigger.
